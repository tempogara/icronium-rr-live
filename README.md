# ICRONIUM RR Live Forwarder

Questo progetto è importabile in Eclipse/IntelliJ/VS Code come **Gradle project**.

## Avvio (macOS/Linux)
```bash
./gradlew run
```

## Avvio (Windows)
```bat
gradlew.bat run
```

> Nota: qui `gradlew` è un launcher minimale che scarica la distribuzione di Gradle indicata in `gradle-wrapper.properties`.
> Non serve avere Gradle già installato.
